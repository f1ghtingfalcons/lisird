LISIRD Tools

TSS_Reader - The tss_reader is an IDL class to read data from LaTiS.

Plot_Solar_Code - The files in the plot_solar_code directory are IDL procedures to read and display the full mission solar irradiance data files (SME, UARS, TIMED, SORCE) from the LASP Interactive Solar Irradiance Data Center (LISIRD).

LaTiS - An API that allows software programs to access most data sets on LISIRD.

LASP Time Series Server (LaTiS)


The LASP Time Series Server (LaTiS) is an API that allows software programs to access most data sets on LISIRD.* LaTiS provides customizable data access functionality. With a LaTiS URL a client can:

  • select a data set
  • specify the desired variables
  • specify the desired range
  • pass the result through a number of filters
  • specify an output format

Specific details about LaTiS are provided below.

*Some data sets are not currently served by LaTiS. Information about their APIs may be found on their respective LISIRD pages.


Example LaTiS URLs:

Spectral data from a single day of UARS, in CSV format:

http://lasp.colorado.edu/lisird/tss/uars_solstice_ssi.csv?wavelength,irradiance&time~2001-01-01

SORCE TSI data for one year

http://lasp.colorado.edu/lisird/tss/sorce_tsi_24hr.csv&time>=2005-01-01T12:00&time< 2006-01-01T12:00

Info page for Historical TSI

http://lasp.colorado.edu/lisird/tss/historical_tsi.html

IDL code to retrieve SORCE MgII data

http://lasp.colorado.edu/lisird/tss/sorce_mg_index.pro

Time Series Server

Time Series Server

Usage: http://lasp.colorado.edu/lisird/tss/dataset.suffix?projection&selection&filter

dataset: Name of dataset (see Available Datasets)
suffix: Type of output (see Output Options)
projection: List of variables to return with optional hyperslab (index subset) definitions. Default to all.
The Dataset Descriptor Structure (DDS) will describe the variables for each dataset. Use the dds suffix to get a dataset's DDS.
selection: Zero or more relative constraints on a variable (e.g. time>=2010-01-01). Each must be preceded by a '&'.
filter: Zero or more filters to be applied to the data (see Filter Options). Each must be preceded by a '&'.

Output Options (suffix)

csv
ASCII comma separated values
dods
Standard OPeNDAP data output as defined by the Data Access Protocol (DAP), binary
html
Web interface to the data set, HTML
dat
Space delimited ASCII tabular output with Sequence variables flattened to a single row.
bin
IEEE 64-bit floats, little-endian
json
JavaScript Object Notation (JSON), ASCII
info
Information about the data set, ASCII
das
Standard OPeNDAP Dataset Attribute Structure (DAS), ASCII
pro
IDL code
dds
Standard OPeNDAP Dataset Descriptor Structure (DDS), ASCII

Filter Options

last
return only the last time sample
thin
thin(n), apply a stride to return about n time samples
replace_missing
replace_missing(a), replace missing values with the value a
exclude_missing
exclude_missing(), exclude any time sample that has a missing value
replace
replace(a,b), replace any occurrence of the value a with b
format_time
format_time(format), format ASCII time output, see Java's SimpleDateFormat.
Examples: format_time(yyyy-MM-dd'T'HH:mm:ss) yields 2012-01-01T23:59:59
format_time(yyyyDDD) yields 2012001
format_time(d MMM yyyy) yields 1 Jan 2012
first
return only the first time sample
stride
stride(n), return every nth time sample
